## Indítás :

## Webszerver szimuláció
        Indítsa el a XAMPP Control Panel alkalmazást.
        Kapcsolja be az alábbi modulokat:
            - Apache
            - MySQL
## Adatbázis importálása:
        MySQL Admingombjára kattintson.
        Bal oldali menüben kattintson az 'új' elemre.
        Az adatbázis nevéhez irja be: 'pizzaparadise'.
        A mellete lévő legördülő menüben válassza ki az 'utf8mb4_general_ci' elemet
        majd kattintson a 'Létrehozás' gombra.
        A felső menüben kattintson az 'Importálás' menüpontra.
        Az oldal első szegmensében talál egy 'Fálj kiválasztása' gombot kattintson rá,
        majd a \pizza_paradise\WEB\db elérési útvanalon található pizzaparadise.sql fájlt válassza ki majd kattintson a megnyitás gombra.
## Weblap inditása:
        Nyissa meg a böngészőt.
        A felső kereseő mezőbe irjabe a következő URL-t:
        http://localhost/"Almappa neve"/pizza_paradise/WEB
        



    Amint mindkét szolgáltatás fut, a projekt használatra kész.

A szolgáltatások elindítása után a rendszer készen áll a használatra.
